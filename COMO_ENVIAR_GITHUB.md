# Como Enviar seu Trabalho via GitHub

Este guia explica como enviar o arquivo `TechMarica_Producao.sql` para o GitHub, conforme solicitado no trabalho.

## Passo 1: Criar uma Conta no GitHub (se não tiver)
1. Acesse [github.com](https://github.com/).
2. Clique em **Sign up** e siga os passos para criar sua conta gratuita.

## Passo 2: Criar um Novo Repositório
1. Faça login na sua conta.
2. No canto superior direito, clique no ícone **+** e selecione **New repository**.
3. Preencha os campos:
   - **Repository name**: `Trabalho-Banco-Dados-TechMarica` (ou outro nome de sua preferência).
   - **Public/Private**: Selecione **Public** (para que o professor possa ver).
   - Marque a opção **Add a README file** (opcional, mas recomendado).
4. Clique em **Create repository**.

## Passo 3: Enviar o Arquivo SQL
1. Dentro do seu novo repositório, clique no botão **Add file** > **Upload files**.
2. Arraste o arquivo `TechMarica_Producao.sql` (que está na pasta `p2`) para a área de upload ou clique em **choose your files** para selecioná-lo.
3. Aguarde o upload terminar.
4. No campo "Commit changes" (abaixo da área de upload), escreva uma mensagem como: "Adicionando script final do trabalho".
5. Clique no botão verde **Commit changes**.

## Passo 4: Enviar o Link
1. Copie o link (URL) da página do seu repositório (ex: `https://github.com/seu-usuario/Trabalho-Banco-Dados-TechMarica`).
2. Envie esse link na plataforma onde o trabalho deve ser entregue.

---
**Dica**: Se você precisar fazer alterações, basta editar o arquivo no seu computador e repetir o Passo 3 para enviar a versão atualizada.
